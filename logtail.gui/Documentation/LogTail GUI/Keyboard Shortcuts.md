
**File Operations:**
- **Ctrl+O** - Open log file dialog
- **F5** - Reload current file

**Navigation:**
- **Page Up**/**Down** - Scroll by page
- **Home** - Jump to first log entry
- **End** - Jump to last log entry

**Dialogs:**
- **Ctrl+F** - Open Filter dialog
- **Ctrl+S** - Open Settings dialog

**Dialog Controls:**
- **Enter** - Apply changes
- **Escape** - Cancel
